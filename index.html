Layout Systems

Flexbox vs Grid

Flexbox is designed for one-dimensional layouts - either a row OR a column. It's perfect for components where you need to align items in a single direction.

CSS Grid is designed for two-dimensional layouts - both rows AND columns simultaneously. It's ideal for complex page layouts where you need precise control over both dimensions.

In this portfolio, I'm using CSS Grid for the main layout sections (projects, about, contact) because it provides better control over complex responsive layouts, while using Flexbox for smaller components like navigation and social links.
